PICPLANR VERSION 37.5 — GOOGLE CALENDAR

Upload the ZIP contents into the root of the GitHub repository.

Run supabase/google-calendar.sql in Supabase SQL Editor.

In Google Cloud:
1. Enable Google Calendar API.
2. Configure the OAuth consent screen.
3. Create Web application OAuth credentials.
4. Add this authorised redirect URI:
https://www.picplanrapp.com/api/oauth/google/callback

Add these Vercel variables:
GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET
GOOGLE_REDIRECT_URI=https://www.picplanrapp.com/api/oauth/google/callback

Keep TOKEN_ENCRYPTION_KEY.

This version:
• connects Google Calendar
• syncs scheduled posts
• adds a 24 hour popup reminder
• links back to PicPlanr
• lets clients edit caption, date and time in PicPlanr
• updates the same Google event after edits
