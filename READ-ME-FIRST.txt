PICPLANR VERSION 40.5 — SAVED ACCOUNT VOICE

WHAT THIS FIX DOES

• Saves the client's account voice, onboarding answers and website analysis.
• Restores them automatically whenever the workspace loads.
• Lets image analysis continue without repeating page one.
• Does not consume another website analysis merely to reuse saved information.
• Uses browser storage as a fallback and Supabase as the permanent workspace record.
• Restores the account voice on another device after login.
• Converts an older saved website analysis into a usable account voice automatically.

FILES TO REPLACE

index.html
api/account/profile.js

SQL TO RUN ONCE

supabase/account-voice-memory.sql

The existing website analysis already completed on this browser can be restored
without running the website analysis again. After deployment and running the SQL,
refresh PicPlanr, upload an image and continue.

A future separate button can be added for customers who deliberately want to
refresh their website analysis. Reusing the saved voice does not count as a new
website analysis.
