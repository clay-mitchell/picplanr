PICPLANR APEX DOMAIN FIX

This fixes the login glitch caused by switching between:
- https://picplanrapp.com
- https://www.picplanrapp.com
- Vercel preview addresses

Replace these exact files in GitHub:
1. index.html
2. api/oauth/instagram/start.js
3. api/oauth/instagram/callback.js

Then set these Vercel Production environment variables:

PICPLANR_APP_URL=https://picplanrapp.com
META_REDIRECT_URI=https://picplanrapp.com/api/oauth/instagram/callback

In Meta Business Login settings, add this exact OAuth redirect URI:
https://picplanrapp.com/api/oauth/instagram/callback

Remove the www version and any vercel.app versions from the Instagram OAuth redirect list.

Redeploy Production, then always open:
https://picplanrapp.com
