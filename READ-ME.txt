PICPLANR VERSION 40.8 — CAPTION USAGE FIX

Replace these files:

index.html
api/picplanr.js
api/_lib/plans.js

WHAT CHANGED

• Automatic caption creation no longer uses a separate caption regeneration allowance.
• Accuracy checking is included in the normal image processing flow.
• The misleading Caption regenerations card has been removed.
• The main usage card now reads Images and captions processed.
• Every analysed image includes its three caption options within the same allowance.
• Starter includes 250 images with captions.
• Business includes 750 images with captions.
• Pro includes 2,000 images with captions.

No Stripe or Supabase changes are required.

Old caption_regeneration usage records can remain in Supabase. The updated plan
definitions ignore them, so they will no longer appear on the dashboard.
