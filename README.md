# PicPlanr Integrated V24 — Instagram and TikTok Profile Previews

This version adds visual profile previews for the Instagram and TikTok handles entered during onboarding.

## Added

- Instagram preview tab
- TikTok preview tab
- platform-specific profile layouts
- supplied name, handle, category, goal and website
- honest placeholder statistics
- placeholder recent-content grids
- uploaded images can appear in the Instagram grid
- uploaded video filenames can appear in the TikTok grid
- tabs automatically adapt to the handles entered
- clear explanation that real profile data requires account connection

## Important

The previews do not invent live profile images, statistics, biographies or content.
Real data appears only after the social account is connected through the official platform connection.
