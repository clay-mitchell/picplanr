# PicPlanr Integrated V19 — British English and Natural Punctuation

This version changes all generated captions and Story text to use British English.

## Changed

- captions use British spellings such as:
  - colour
  - organise
  - personalised
  - centre
  - favourite
- em dashes and en dashes are blocked
- spaced hyphens are removed from captions
- commas, full stops and separate sentences are used instead
- a final browser clean-up layer catches unwanted punctuation or American spellings
- Story text follows the same rules

- `public/index.html`
- `public/styles-v19.css`
- `public/app-v19.js`
- `api/picplanr.js`
